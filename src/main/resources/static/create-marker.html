📋 마커 정보

크기: 40x50 픽셀

색상: 밝은 청록색/터키스 그라데이션

모양: 핀/물방울 형태

내부 요소: 검은색 테두리 + 청록색 채움 사각형

📥 다운로드 단계

  1. 아래 "마커 이미지 생성" 버튼을 클릭하세요
  2. 브라우저가 자동으로 custom-marker.png 파일을 다운로드합니다
  3. 다운로드된 파일을 src/main/resources/static/images/ 폴더에 저장하세요
  4. 파일명을 custom-marker.png로 유지하세요

👀 미리보기

💡 참고사항: 이미지가 생성되면 자동으로 다운로드됩니다. 다운로드된 이미지를 프로젝트의 images 폴더에 저장한 후, map.js와 admin.js에서 저장된 이미지 링크를 사용할 수 있습니다.
